# Customer Churn Prediction & Retention Dashboard

## Project Overview

This project analyzes telecom customer data to predict churn risk and develop retention strategies. The workflow combines Python machine learning models and Power BI dashboards.

The objective is to identify high-risk customers and provide actionable business insights.

---

## Tools Used

- Python
- Pandas
- Scikit-learn
- Power BI
- Data Visualization

---

## Machine Learning Models

Two models were trained to predict churn:

- Logistic Regression
- Random Forest

Model performance was evaluated using:

- Accuracy
- ROC-AUC
- Classification Report

## Key Insights

- Month-to-month contracts have the highest churn risk
- Customers with fewer services are more likely to churn
- Higher monthly charges correlate with churn probability
